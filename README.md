# Magnetocaloric-Effect

From Isotherm M(H) curves, using The Maxwell Relation You Can Calculate The Entropy Change at Those Specific Temperature and also can plot ∆S vs Temperature Graph with the help of this program. Click https://youtu.be/u_hVE2ihys0 to know how it works.

At first create a blank xlsx file, where you want to extract the ∆S data and of course close the file. If not the program will not get access to this xlsx file.
